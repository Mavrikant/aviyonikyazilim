# MISRA C Kılavuzları

| Künye | |
|----|----|
| Yayımcı | MISRA (The MISRA Consortium) |
| Güncel sürüm | MISRA C:2012 ve sonrası düzeltme/genişletmeler (MISRA C:2023 birleştirilmiş metin) |
| Tür | C dili alt küme kuralları (kılavuz doküman) |
| Kimin için | Emniyet-kritik projede C yazan/denetleyen herkes |

## Neden edinmeli?

C, aviyonikte en yaygın dildir; ama tanımsız davranışları ve zayıf tip denetimiyle
emniyetli bir dil değildir. MISRA C, dilin tehlikeli özelliklerini kural kural
yasaklayarak veya kısıtlayarak öngörülebilir bir alt küme tanımlar. Kurallar
gerekçeleriyle birlikte verilir; çoğu statik analiz aracıyla otomatik denetlenir.
Otomotiv kökenli olmasına karşın havacılık dahil tüm emniyet-kritik sektörlerde
fiilî C kodlama standardı temelidir.

Projede tipik kullanım: MISRA C temel alınır, proje kodlama standardı bunun
üzerine hangi kuralların zorunlu olduğunu ve sapma sürecini tanımlar. Kılavuz
[MISRA'dan](https://misra.org.uk/product/misra-c2023/) (misra.org.uk) ücretli PDF olarak edinilir.
